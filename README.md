# pyGame River Raid
Recreation of River Raid in Python, the famous Atari 8-bit computers' game, my first love ever since I was 7 years old 
![gameplay](rsc/preview.gif)

## Requirements
```
Python >= 3.7.0
pygame >= 2.1.2
```

## Installation
```
pip install pygame 
```

## Running the game
```
python game.py
```
